# AI Skills

AI Skills are reusable instruction packs stored in Memside. They are useful for structured guidance that should be reused across AI tools, such as repository instructions, review checklists, writing rules, or project-specific workflows.

AI Skills are separate from normal memories. Memories store facts, notes, tasks, decisions, and checkpoints. AI Skills store instructions that tell an AI how to work.

## Supported Sources

AI Skills are useful for importing sources such as:

- `SKILL.md`
- `AGENTS.md`
- `CLAUDE.md`
- Markdown rule files
- plain text instruction files
- Memside AI Skill exports

Memside preserves the original source so the imported instruction can be reviewed later.

## Compact and Detailed Versions

An AI Skill can have optimized forms. A compact version is useful when the AI needs the core instruction with fewer tokens. A detailed version is useful when fidelity matters more than size.

This gives you a practical way to keep the original source intact while still giving connected AI tools a context-efficient version when appropriate.

## How Connected AI Tools Use Skills

Connected AI tools can list available skills, fetch a selected skill, inspect the original source, or use the optimized version. This is useful when one AI tool creates or improves a workflow and another AI tool needs to use the same workflow later.

Activation remains a user-controlled step. A skill can be imported, improved, reviewed, and only then activated.
