# User AI Profile

The User AI Profile stores reusable personal guidance for connected AI tools. It is for preferences that should travel with you across tools, projects, and sessions.

A good profile can describe how you prefer technical answers, how much detail you want, what tradeoffs matter to you, how you like plans structured, and which linked memories are especially important.

## Why It Helps

Most AI tools start each chat without knowing how you work. You may prefer direct answers, careful verification, minimal changes, or a specific writing style. Without a profile, you repeat those preferences often.

The User AI Profile gives connected AI tools a compact way to understand your working style. It can reduce repeated setup text and make handoffs between AI tools smoother.

## Linked Memories

The profile can link a small number of important memories. Linked memories help the AI discover related context without putting the full content of every memory into every startup packet.

Use linked memories for stable preferences, important operating rules, or high-value background that should be easy to find. Secret memories should not be linked to the profile.

## Practical Example

You might use the profile to say that you prefer concise engineering answers, focused verification, and clear separation between implementation work and product planning. You can then link operating rules or project memories that explain those preferences in more detail.

When a connected AI tool reads the profile, it can use that guidance without you restating it every time.
