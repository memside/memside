# Support

For product support, email:

```text
support@memside.com
```

When reporting a connection issue, include the AI tool you are using, the setup method, and the error text you see. Screenshots help, but please avoid sending private memory contents unless we ask for a specific example.

Useful details include:

- AI tool or client name
- whether you used OAuth or API-key auth
- the MCP server URL you entered
- browser and device if the issue happens during login
- approximate time of the failed attempt

## Disconnecting Access

If you no longer want an AI tool or script to access Memside, remove the OAuth connector in that tool's app or connector settings. If the connection uses a Memside API key, revoke the key in Memside settings.

If you think a credential was exposed, revoke or disconnect it before sending a support request. Do not paste real API keys, OAuth tokens, passwords, or private memory contents into public issues.

For security reports, use [SECURITY.md](SECURITY.md).
