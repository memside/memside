# Contributing

This repository is open for public documentation, examples, and integration feedback. It is not the place for private production source or deployment configuration.

Good contributions include clearer setup steps, improved examples, typo fixes, troubleshooting notes, and public-safe feedback about MCP or API-key usage. Keep examples small and focused so new users can copy them without reading a full application.

Before opening a pull request, make sure the change does not include secrets, private URLs, internal architecture, screenshots with personal data, or details from a private Memside account.

## Writing Style

Write plainly. Prefer short sections, concrete examples, and complete setup steps. Avoid marketing language where a technical explanation would be more useful.

## Issues

Use issues for public docs problems, example bugs, or setup confusion. Security issues should be reported privately through [SECURITY.md](SECURITY.md).
